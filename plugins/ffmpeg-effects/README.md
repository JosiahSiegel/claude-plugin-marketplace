# Ffmpeg Effects Plugin

Creative FFmpeg filtergraph effects for motion graphics and visual treatments. PROACTIVELY activate for transitions, xfade, color grading, chromakey/green screen, drawbox/drawtext overlays, shapes, lower thirds, glitch/datamosh/VHS looks, karaoke lyrics, kinetic captions, animated text, waveform/spectrum visualization, and animation timing.

## Installation

```bash
claude plugin add ffmpeg-effects
```

## Components

This focused plugin contains the agents, skills, commands, scripts, or references relevant to `ffmpeg-effects`.
