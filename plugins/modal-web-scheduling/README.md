# Modal Web Scheduling Plugin

PROACTIVELY activate for FastAPI endpoints, @modal.fastapi_endpoint, ASGI/WSGI apps, webhooks, custom domains, WebSockets, long-running APIs, cron schedules, modal.Cron, modal.Period, scheduled ETL, deployment behavior for schedules, and endpoint debugging.

```bash
claude plugin add modal-web-scheduling
```
