# Connected TV (CTV) Optimization

## CTV Growth Statistics (2025-2026)

| Statistic | Value |
|-----------|-------|
| YouTube position | Most-watched streaming service on TVs in US |
| Surpassed | Netflix |
| Daily video consumption | 1 billion hours |
| CTV performance | Long-form outperforms other formats |

**Key Insight**: MrBeast reports stronger CTV performance for long-form content.

## Why CTV Matters

### Viewing Behavior Differences

| Behavior | Mobile | CTV |
|----------|--------|-----|
| Session length | Short bursts | Extended viewing |
| Attention | Divided | Focused |
| Skip tendency | High | Lower |
| Content preference | Short-form | Long-form |
| Audio | Often muted | Full volume |

### CTV Viewer Expectations

- Higher production value
- Longer, in-depth content
- Documentary-style formats
- Series and episodic content
- Professional audio quality

## Visual Optimization for CTV

### Resolution Requirements

| Resolution | Recommendation |
|------------|----------------|
| Minimum | 1920x1080 (1080p) |
| Recommended | 3840x2160 (4K) |
| HDR | Consider for premium content |

### Text and Graphics

| Element | Mobile | CTV |
|---------|--------|-----|
| Text size | 30pt+ | Can be slightly smaller |
| Graphic detail | Simple | More detail visible |
| Lower thirds | Large, bold | Standard size works |
| Subtitles | Essential | Optional (audio on) |

### Composition Considerations

- Wider shots work better on large screens
- More detail in background
- Cinematic aspect ratios appreciated
- Professional color grading expected

## Audio Optimization for CTV

### Speaker System Considerations

CTV viewers often have:
- Sound bars
- Surround sound systems
- High-quality speakers

### Audio Best Practices

| Element | Recommendation |
|---------|---------------|
| Quality | 48kHz, stereo minimum |
| Dynamic range | Balanced for speakers |
| Voice clarity | Crystal clear |
| Music | High quality, properly mixed |
| Sound effects | Appropriate levels |

## Content Formats for CTV

### High-Performing Formats

| Format | Why It Works on CTV |
|--------|-------------------|
| Documentary | In-depth, lean-back viewing |
| Educational deep dives | Extended attention span |
| Interviews | Conversational, engaging |
| Series | Binge-watching behavior |
| Tutorials | Dedicated learning time |

### Optimal Length

| Duration | CTV Performance |
|----------|-----------------|
| 10-20 minutes | Good |
| 20-40 minutes | Very good |
| 40-60 minutes | Excellent |
| 60+ minutes | Premium long-form |

CTV viewers accept and expect longer content.

## Chapters for CTV Navigation

### Why Chapters Matter for CTV

- TV remote navigation
- Easy section jumping
- Resume watching
- Preview content structure

### Chapter Best Practices

```
0:00 Introduction
2:30 Part 1: [Topic]
8:45 Part 2: [Topic]
15:20 Part 3: [Topic]
22:00 Conclusion
```

- Clear, descriptive titles
- Natural break points
- Enable easy navigation
- Show up in TV interface

## Series and Episodic Content

### Why Series Works on CTV

- Binge-watching culture
- Playlist auto-play
- Return viewer behavior
- Algorithm rewards consistency

### Series Structure

| Element | Recommendation |
|---------|---------------|
| Episode length | Consistent (e.g., all 20-30 min) |
| Release schedule | Regular (weekly) |
| Episode naming | Clear numbering |
| Playlist | Organized, ordered |
