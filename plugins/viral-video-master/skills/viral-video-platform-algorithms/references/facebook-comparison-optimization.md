## Facebook Reels Algorithm

### Algorithm Signals

| Signal | Weight | Description |
|--------|--------|-------------|
| Shares | Very High | Facebook's core mechanic |
| Comments | High | Discussion generation |
| Watch Time | High | Completion rate |
| Reactions | Medium | Emotional responses |
| Saves | Medium | Bookmark behavior |

### Facebook Algorithm Priorities

**Content Preferences:**
- Share-worthy content (highest priority)
- Conversation-generating posts
- Native content (not cross-posted quality issues)
- Vertical video format (70% brand lift)

### Facebook Posting Schedule

| Time | Performance |
|------|-------------|
| Late Friday night | Highest views |
| Midnight into Saturday | Peak performance |
| First 1-2 hours | Critical for reach |

**Key Insight:**
Facebook Reels timing is notably different from other platforms.

### Facebook vs Instagram Cross-Posting

**When to Cross-Post:**
- Content fits both audiences
- No platform-specific elements that don't translate
- Quality maintained across platforms

**When Not to Cross-Post:**
- Content too casual for Facebook audience
- Instagram-specific features used
- Audience preferences significantly differ

## Algorithm Optimization Framework

### Universal Optimization Strategy

1. **Hook Optimization** (All Platforms)
   - 1.3-3 seconds critical
   - Pattern interrupt + curiosity

2. **Retention Optimization** (All Platforms)
   - Minimize drop-offs
   - Pattern interrupts throughout
   - No filler content

3. **Engagement Optimization** (Platform-Specific)
   - Encourage comments
   - Add share triggers
   - Include clear CTAs

4. **Discovery Optimization** (Platform-Specific)
   - Hashtag strategy
   - Keyword optimization
   - Trending sound usage

5. **Consistency** (All Platforms)
   - Regular posting schedule
   - Content category focus
   - Quality maintenance
