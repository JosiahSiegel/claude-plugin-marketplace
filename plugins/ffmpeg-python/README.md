# Ffmpeg Python Plugin

Python FFmpeg integration for frame pipelines, PyAV/libav bindings, OpenCV handoff, and validation helpers. PROACTIVELY activate for PyAV install/API usage, av.open, VideoFrame/AudioFrame, ffmpeg-python parameter mapping, subprocess command generation, rawvideo pipes, BGR/RGB bugs, OpenCV/cv2 processing, ffmpegcv, VidGear, Decord, and ML video pipelines.

## Installation

```bash
claude plugin add ffmpeg-python
```

## Components

This focused plugin contains the agents, skills, commands, scripts, or references relevant to `ffmpeg-python`.
